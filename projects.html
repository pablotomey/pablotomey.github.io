Proyectos mobile y web

Apprender

Desarrollo de prototipo de aplicación móvil en Android nativo, orientada a la enseñanza de la lectura para personas adultas con problemas de alfabetización. Sus principales funciones son el reconocimiento de voz para la interacción del usuario e instrucciones de audio para las instrucciones.

Covid Monitor

Aplicación móvil desarrollada en Android nativo la cual entrega datos de los contagios por Covid-19 en tiempo real en Chile. Esta información se despliega de forma general y por región, además de consultar datos históricos. Esta aplicación consume datos de la API 'Chile-coronapi' a través de la libreria Retrofit.

Apprender landing page

Landing page informativa para el proyecto móvil Apprender. Pagina de aterrizaje que muestra las principales características de este proyecto tecnológico como también una sección de contacto, cuenta con un diseño full responsive.

Portafolio web

Portafolio encargado de mostrar información relevante sobre capacidades técnicas y proyectos realizados a lo largo del tiempo. Diseño full responsive y con diseños vectoriales realizados con Adobe Illustrator.

Evaluación web app

Aplicación web Responsiva para realizar evaluaciones nutricionales a los pacientes del área de nutrición del Hospital Clínico Magallanes, XII Región de Chile. Cuenta con el registro y administración de cuentas de usuario, registro y administración de pacientes, registro de Evaluación nutricional con cálculos automáticos de varios parámetros requeridos como el IMC entre otros.
